# @rovahub/vue-float-chat

This Vue2 plugin injects float chat instance into all of your vue instances

## Install
``` bash
# Npm
npm install @rovahub/@rovahub/vue-float-chat --save

# Yarn
yarn add @rovahub/@rovahub/vue-float-chat
```
